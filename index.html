Quick Setup

Running in four steps.

01
Install Dependencies

Install required Python packages via pip.

pip install google-auth google-auth-oauthlib google-api-python-client serpapi
02
Configure Credentials

Add your ValueSERP key and Google Sheets credentials to the config section.

# In seo_rank_tracker_valueserp.py VALUESERP_API_KEY = "your_valueserp_key_here" SPREADSHEET_NAME = "Your Google Sheet Name" CREDS_FILE = "credentials.json"
03
Prepare Your Sheet

Ensure column A contains your keywords. The tracker reads from row 2 onward and writes rankings into adjacent columns.

# Sheet layout expected: A: Keyword | B: Position | C: URL | D: Date
04
Run the Tracker

Execute the script and watch rankings flow into your Sheet in real time.

python3 seo_rank_tracker_valueserp.py # To resume an interrupted run: # Just run the same command — checkpoint is auto-detected
